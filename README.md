## Angular with requirejs & bower & grunt

## Usage

1. make sure you had installed `node`, `ruby`, `bower`, `grunt`.

2. cd `root-folder`, then `npm install && bower install && bundle install`.

3. to develop, cd `root-folder`, then `grunt server`.
